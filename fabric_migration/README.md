# Microsoft Fabric Migration Workspace

Folder ini didedikasikan untuk menyimpan skrip dan data eksperimen selama proses migrasi ke Microsoft Fabric.

Codebase utama di luar folder ini (dashboard HTML, JS, dan skrip asli) tidak akan disentuh, sehingga Anda tetap memiliki versi lokal yang aman untuk di-deploy kapan saja jika migrasi ini gagal.
